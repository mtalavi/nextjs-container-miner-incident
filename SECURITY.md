# Security Policy

## Scope

This repository contains an anonymized defensive write-up and hardening guidance. It is not a live service or application.

## Safe discussion

Please keep discussions defensive and anonymized.

Do not include private infrastructure details, secrets, production logs, or information that identifies a live system.

## Defensive-only intent

The repository is intended to help developers, founders, and DevOps teams harden containerized applications. Contributions should remain defensive, anonymized, and non-operational.
